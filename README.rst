Installation
------------

At the command line either via easy_install or pip:

.. code-block:: shell

    $ pip install cache_requests

.. code-block:: shell

    $ easy_install cache_requests

**Uninstall**

.. code-block:: shell

    $ pip uninstall cache_requests


Getting Started
---------------
To use cache_requests in a project:

.. code-block:: python

    import cache_requests
